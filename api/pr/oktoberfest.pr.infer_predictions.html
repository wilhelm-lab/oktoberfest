oktoberfest.pr.infer_predictions

oktoberfest.pr.infer_predictions(triton_client, model, input_data, outputs, batch_size)

Infer predictions from a triton client.

Parameters:

  • triton_client (InferenceServerClient) – An inference client using grpc

  • model (str) – a model that is recognized by the server specified in the triton client

  • input_data (Dict[str, Tuple[ndarray, str]]) – a dictionary that contains the input names (key) for the specific model and a tuple of the input_data as a numpy array of shape [:, 1] and the dtype recognized by the triton client (value).

  • outputs (List[str]) – a list of output names for the specific model

  • batch_size (int) – the number of elements from the input_data that should be provided to the triton client at once

Return type:

Dict[str, ndarray]

Returns:

a dictionary containing the predictions (values) for the given outputs (keys)
