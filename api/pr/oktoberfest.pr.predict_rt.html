oktoberfest.pr.predict_rt

oktoberfest.pr.predict_rt(data, **kwargs)

Retrieve retention time predictions from koina and add them to the provided data object.

This function takes a dataframe containing information about PSMS and predicts retention time using a koina server. The configuration of koina is set using the kwargs.

Parameters:

  • data (AnnData) – Anndata object containing the data required for prediction and to store the predictions in after retrieval from the server.

  • kwargs – Additional keyword arguments forwarded to Koina::predict
