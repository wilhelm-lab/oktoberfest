__init__(model_name, server_url='koina.wilhelmlab.org:443', ssl=True, targets=None)

Initialize a KoinaModel instance with the specified parameters.

This constructor initializes the KoinaModel instance, connecting it to the specified Inference Server. It checks the availability of the server, the specified model, retrieves input and output information, and determines the maximum batch size supported by the model’s configuration. Note: To use this class, ensure that the inference server is properly configured and running, and that the specified model is available on the server.

Parameters:

  • model_name (str) – The name of the Koina model to be used for inference.

  • server_url (str) – The URL of the inference server. Defaults to “koina.wilhelmlab.org:443”.

  • ssl (bool) – Indicates whether to use SSL for communication with the server. Defaults to True.

  • targets (Optional[List[str]]) – An optional list of targets to predict. If this is None, all model targets are predicted and received.

  • disable_progress_bar – Whether to disable the progress bar showing the progress of predictions.
