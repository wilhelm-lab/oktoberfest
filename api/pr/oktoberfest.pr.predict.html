oktoberfest.pr.predict

oktoberfest.pr.predict(data, *args, **kwargs)

Retrieve predictions from koina.

This function takes a dataframe containing information about PSMS and predicts peptide properties using a koina server. The configuration of koina is set using the kwargs. See the koina predict function for details. TODO, link this properly.

Parameters:

  • data (DataFrame) – Dataframe containing the data for the prediction.

  • args – Additional positional arguments forwarded to Koina::predict

  • kwargs – Additional keyword arguments forwarded to Koina::predict

Return type:

Dict[str, ndarray]

Returns:

a dictionary with targets (keys) and predictions (values)
